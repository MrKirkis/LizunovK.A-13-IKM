# Физический симулятор

Консольное приложение на C++ для моделирования физических процессов с использованием численных методов и аналитических решений.

## Разработчик
Лизунов Кирилл Андреевич ИТ-13

## Описание проекта

Программа позволяет проводить симуляции четырех физических моделей:
1. **Падение тела с сопротивлением воздуха** - движение тела под действием силы тяжести и силы сопротивления
2. **Колебания пружинного маятника** - гармонические колебания груза на пружине
3. **Движение с постоянным ускорением** - равноускоренное движение
4. **Движение заряда в электрическом поле** - движение заряженной частицы в поле

## Требования

- Компилятор C++ с поддержкой C++11 или выше
- CMake 3.10 или выше (рекомендуется)

## Структура проекта

main.cpp

```

```

##Математические методы
1. Численное интегрирование
Используется метод Эйлера для решения дифференциальных уравнений:

Падение тела: dv/dt = g - (k/m)·v

Пружинный маятник: dv/dt = -(k/m)·x, dx/dt = v

2. Аналитические формулы
Для постоянного ускорения:

Позиция: x(t) = x₀ + v₀t + at²/2

Скорость: v(t) = v₀ + at
